MVP de un Panel de Administrador para la gestión de menús de un restaurante. El objetivo es permitir al staff administrativo dar de alta, editar y visualizar el catálogo de platillos.

Requerimientos funcionales:

Dashboard principal con resumen de categorías.

Formulario para creación de platillos (campos: nombre, descripción, precio, categoría y estado de disponibilidad).

Tabla de visualización de productos con opciones de búsqueda y filtrado.

Stack Técnico:

Framework: React.

Estilos: Tailwind CSS.

UI Library: Componentes de shadcn/ui (específicamente Table, Form, Input, Button y Card).

El diseño debe ser limpio, profesional y responsivo, priorizando la facilidad de uso para el usuario administrativo."
